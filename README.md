# Getting started

## Запуск проекта

```
npm install - устанавливаем зависимости
npm start - Запуск в dev-режиме
npm run build:prod - Сборка в prod-режиме
```

_Готовая сборка вот тут -_ https://chesscheckers.netlify.app/game

---

В проекте использовал:

-  React
-  Redux/toolkit
-  FSD - архитектура
-  Webpack
-  TypeScript
-  и другое.
